## ⚙️ How to Run Any Example

1. **Clone the repo**
   ```bash
   git clone https://github.com/MuhammadWaris-55/Backend.git
   ```

2. **Go into a topic folder**
   ```bash
   cd Basic Server
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the server**
   ```bash
   node index.js
   # or
   npx nodemon index.js
   ```
